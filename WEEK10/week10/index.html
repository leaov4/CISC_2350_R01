Multiple Backgound Images

The background image property in CSS applies a graphic or a gradient to the background of an element. Backgound-image also allows for multiple background images to overlap on the same element. The stacking order places the image that should be at the front first, and the image that should be at the back last.


Background-position places the image within the element, the position for the sun to the right is left-top. We can also set background-repeat to either no-repeat, (the sun to the left and the sun above) or set it to repeat (the row of suns at the bottom of the page).
